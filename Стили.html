Свойства CSS

1. Сolor - изменяет цвет текста
color: blueviolet;

текст

2. Background-color — устанавливает цвет фона для элемента
background-color: blueviolet;

Текст

3. Font-size - указывает размер для шрифта
font-size: 40px;

Текст

4. Font-family - указывает шрифт или семейство шрифтов для текста
font-family: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif;

Текст

5. Font-style - изменяет стиль шрифта
font-style: italic;

Текст

6. Text-align - изменяет выравнивание текста
text-align: right;

Текст

7. Text-shadow - создаёт тень тексту
text-shadow: 3px 3px 3px blueviolet;

Текст

8. Border - изменяет/создает толщину границ
border: 10px solid blueviolet;

Текст

9. Padding - устанавливает расстояние (его также называют "внутренний отступ") между внутренним краем рамки элемента и его содержимым
padding-right: 100px;

Текст
